# ⭐ Sugar Blocks(슈가블럭스) - 통합 패키지

컴퓨터 1대로 **게임 배포 + 온라인 대전 서버**가 모두 됩니다.
폰·태블릿·다른 컴퓨터가 이 서버에 접속해서 **서로 크로스플레이 대전**할 수 있어요.

---
## 1. 컴퓨터 서버 설치 (딱 3단계)

1) **Node.js 설치** (한 번만): https://nodejs.org 접속 → "LTS" 초록 버튼 → 다운로드 후 계속 "다음"만 누르면 끝.
2) 이 폴더를 원하는 곳에 두고:
   - **윈도우**: `start-server.bat` 더블클릭
   - **맥/리눅스**: 터미널에서 `./start-server.sh`
3) 검은 창에 이렇게 뜨면 성공!
   ```
   ⭐ 별사탕 테트리스 서버 시작!
      게임 열기: http://localhost:8765
      폰에서:    http://192.168.x.x:8765   ← 이 주소를 폰에 입력!
   ```

- 내 컴퓨터에서 게임하기: 브라우저에 `http://localhost:8765`
- 같은 와이파이의 폰에서: 위에 표시된 `http://192.168.x.x:8765`
- 게임 안 로비의 서버 주소는 **자동으로 채워집니다** (이렇게 열었을 때)
- 방화벽 창이 뜨면 "허용"을 눌러주세요.

## 2. 인터넷 공개 (무료 호스팅)
같은 와이파이가 아닌 친구와 하려면 무료 호스팅에 이 폴더를 올리면 됩니다.
(README 하단 목록 참고 — 올리면 `https://내앱주소` 로 전세계 접속!)
호스팅에 올린 경우 게임 로비 주소는 `wss://내앱주소` 형태가 자동 적용됩니다.

## 3. 파일 설명 (관리자용)
| 파일 | 역할 | 교체 포인트 |
|---|---|---|
| index.html | 게임 전체 | 상단 `SOUND_FILES`(효과음 mp3), `BACKGROUNDS`(배경 그림 BG-01~10), `REMOTE_BG_URLS` |
| server.js | 게임 배포 + 대전 중계 + 주간 랭킹 | `PORT` 환경변수로 포트 변경 |
| leaderboard.json | 주간 랭킹 저장 (자동 생성) | 삭제하면 초기화 |
| manifest.json / sw.js / icon-*.png | 📱 폰 앱(PWA) 설치용 | 아이콘·이름 교체 가능 |
