🛠 슈가 블럭스 EDIT

게임의 모든 조정 가능한 값을 개발자가 직접 편집하는 툴입니다.
사용법: ① 아래 값들을 원하는 대로 수정 → ② [📤 배포용 복사] 클릭 → ③ index.html에서 DEPLOY_TUNING을 검색해 null 자리에 붙여넣기 → ④ GitHub 커밋 → Render 자동 배포 → 모든 사용자에게 적용!
※ [💾 브라우저에 임시 저장]을 누르면 이 툴을 다시 열었을 때 작업하던 값이 복원됩니다.

⭐ 뽑기 / 경제

뽑기(갓차) 비용을 조절합니다. 값을 낮추면 아이템이 자주 나와 캐주얼해지고, 높이면 전략성이 올라갑니다.

⏸ 게임 흐름

일시정지·승패 후 대기 시간을 조절합니다.

🔥 콤보 추가 공격

연속으로 줄을 지우면(콤보) 상대에게 추가 공격이 나갑니다. 공식: 추가 공격 줄 수 = 콤보 수 − 아래 값. 기본 1이면 2연속=1줄, 3연속=2줄… 값을 2로 올리면 3연속부터 1줄씩(더 어렵게).

⚔️ 공격 버튼

파이트+아이템 모드의 공격 버튼 동작을 설정합니다. 자동=공격 가능해지면 즉시 발사, 수동=버튼을 눌러야 발사(사용자가 게임 중 버튼을 위/아래로 끌어 직접 바꿀 수도 있음).

🍬 아이템 지속 시간 (ms)

각 효과의 지속 시간입니다. 1000 = 1초. (모든 값은 이미 v17에서 +2초 반영된 기본값입니다)

🎲 아이템 등장 확률 (가중치)

숫자가 클수록 자주 나옵니다 (전체 대비 비율). 0으로 하면 그 아이템은 등장하지 않습니다. 비워두면 게임 기본값 사용.

📝 설명창 (블럭 아래 안내 문구)

게임 화면 하단 설명창의 기본 투명도와 글씨 크기입니다. 사용자도 설정에서 개별 조절할 수 있으며, 여기 값은 "처음 값"입니다.

🎮 이동 버튼

화면 터치 이동 버튼의 기본 크기입니다. (기본 표시 버튼: Z·Space·C·X, 나머지는 사용자가 설정에서 켤 수 있음)

🖼 배경 도감

클래식 모드에서 공개되는 배경 그림 목록입니다. id는 도감에 표시되는 짧은 이름, src는 이미지 주소(URL 또는 데이터URI)입니다. 한 줄이라도 입력하면 게임의 기본 도감 전체가 이 목록으로 교체됩니다. 비워두면 기본 도감 유지.
